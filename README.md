## 使用说明
    想要运行这套代码，您需要将它们拖到garspnet官方文档根目录下。
    建议 torch==1.8.0 numpy==1.23.5 
    视觉模块 cv_process.py 可单独运行，需要安装最新的 ultralytics，并且还需要：
    pip install open-clip 通过这种方式安装clip。

    grasp_process.py 也可单独运行，用来测试你的graspnet是否可以正常运行。
